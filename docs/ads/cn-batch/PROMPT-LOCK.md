# 概念图构图锁定 v2（对齐 Fei-Away gallery，并修白框问题）

## 结构（Codex 壳，不是官网）

1. **不要**整条灰白系统菜单栏（Apple / File / Edit / View / Help）。最多右上角极小窗控，或干脆不要顶栏。  
2. 左侧栏 + 主区氛围图 + **四张建议卡** + **底部输入框**。  
3. 侧栏「项目」写 ProvDex 广告：`ProvDex多智能体中心` / `一键更换皮肤` / `一键切换模型` / `多工具统一入口`。壳品牌仍可写 Codex。

## 视觉（必须）

| 项 | 要求 |
|----|------|
| 侧栏 | **跟主题色染色** + **半透明磨砂玻璃**，能隐约透出背景；禁止死白实心条 |
| 建议卡 | **偏小**、圆角、**半透明玻璃**（粉/金/霓虹淡染），禁止大块不透明白砖 |
| 输入框 | 半透明玻璃条，别做成厚重白广告牌 |
| 装饰 | 主题必需点缀：花瓣/金币/拍立得/贴纸/爱心别针/光斑等（对照仓 skin-01 量级） |
| 比例 | 严格 16:9，禁止拉伸变形 |
| 人物 | 原创角色；勿真人明星 / 明确版权 IP |
