# Source notes — 经典蓝 2007

## What the viral screenshots are

Screenshots titled **「Codex 2007 - 经典蓝工作台」** are a QQ 2007 / Windows XP Luna
chrome restyle of **OpenAI Codex Desktop**. They are **not** an official OpenAI theme.

Typical signals in those shots:

- XP-style glossy blue title bar and window controls
- Messenger-like friend rail / 「Codex 小蓝」 companion panel
- Injector filenames such as `dream-skin.css`, `renderer-inject.js`

## Closest open-source projects

| Project | Role |

| [angziii/codex-qq-skin](https://github.com/angziii/codex-qq-skin) | Dedicated macOS CDP skin with `qq-classic` / `qq-2007-full` (+ MSN/ICQ/Yahoo era packs). Matches the screenshot language most closely. |
| [Fei-Away/Codex-Dream-Skin](https://github.com/Fei-Away/Codex-Dream-Skin) | Popular background/theme injector used by many custom skins (including ProvDex vendor + this hub). |
| [CodeDrobe](https://github.com/CodeDrobe/core) | Alternate portable theme runtime / skill toolchain. |

No public package named exactly `Codex-2007-Classic-Blue` was found as a standalone
gallery preset; the folder name often appears as a **local working tree** while
building on top of Dream Skin / QQ-skin engines.

## This hub preset

`preset-classic-blue-2007` is an **Agent Skin Hub** glass-shell atmosphere pack
(wallpaper + colors + concept preview) inspired by that era. It does **not** ship
the full friend-list chrome engine — for that, use `angziii/codex-qq-skin`.
